# AgentBridge 文档：项目成长编年史 / docs

本目录按**真实开发时间顺序**编排——每个序号是项目成长的一个阶段，从最初的双向桥，到今天的跨网多人多仓协作系统规格。顺着 `01 → 09` 读下来，就能看到 AgentBridge 是怎么一步步长起来的。

> **编排规则**：序号 = 开发先后（01 最早、09 最新）。`01–07` 是对各阶段的**回顾性总结**（综合当时的 commit + 设计文档精华重写）；`08`（v2 架构愿景）、`09`（v3 协作系统规格）是仍在指引开发的**活规格**，保留完整原文 + 阶段定位导言。
>
> **面向用户的叙述一律中文**；代码 / 路径 / 标识符保留原文。
